# 📊 Bookstore Analytics with SQL

## 🔹 Overview

This project analyzes bookstore data using SQL to extract business insights like sales performance, customer behavior, inventory management, and marketing ROI.

## 🧰 Tools Used

* SQL (SQLite)
* Joins, Aggregations, CTEs

## 📂 Database Structure

* Books
* Customers
* Orders
* MarketingSpend

## 📈 Key Insights

* Identified best-selling books and revenue drivers
* Detected low inventory stock
* Performed RFM customer segmentation
* Evaluated marketing ROI
* Found returning and churned customers

## 🚀 How to Run

1. Open SQLite / DB Browser
2. Run `bookstore.sql`
3. Execute queries step-by-step
